== Description ==

Provides additional REST-API endpoints for WooCommerce to enable the ability to add, view, update and delete items from the cart.

== Installation ==

Installing "WooCommerce Cart REST API" can be done using the following steps:

1. Download the plugin via SebastienDumont.com.
2. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress.
